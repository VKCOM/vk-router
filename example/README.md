# Начало работы


- `git clone ...`
- `yarn install`
- Идём по [этой ссылке](http://astolear.cs7777.vk.com/1igfedorov.php?act=vkui_access_token&name=education) и получаем токен приложения
- `yarn start`
- Открываем URL (`http://localhost:10888`, например)
- В консоли вводим `localStorage.setItem('education:access_token', '...')` и подставляем ваш токен
- Перезагружаем страницу
- Вы великолепны
